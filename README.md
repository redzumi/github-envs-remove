# github-envs-remove
Remove your all github repository deployments and environments

## Installation
```
npm install -g github-envs-remove
# or
npm install -g https://github.com/redzumi/github-envs-remove
```

## Usage
```
github-envs-remove

# and follow for simple questions ;)
```
